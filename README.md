## 介绍
cc-ui是uni-app生态组件库, 宗旨是一切为了简单。

因为作者儿子名字叫岑岑, 所以为组件库名字取名为cc, 算是送给儿子的礼物。

本组件库使用与uni-app开发, 目前适配了全部h5和微信小程序端, 大部分的app端, 后续会陆续适配所有端。
在维护此组件库的同时, 作者也会陆续推出vue3版本和react版本, 敬请期待。
